사용자 로그인 기능
================

1. 이메일과 비밀번호로 로그인
2. 로그인 성공 시 메인 페이지로 이동
3. 로그인 실패 시 에러 메시지 표시
4. 자동 로그인 기능 (체크박스)
5. 비밀번호 찾기 링크 제공
6. 소셜 로그인 추가 (구글, 페이스북)

API 엔드포인트:
- POST /api/login
- POST /api/logout
- GET /api/user/profile
- POST /api/social-login

데이터베이스 테이블:
- users (id, email, password, created_at, social_id)
- sessions (id, user_id, token, expires_at)
- social_accounts (id, user_id, provider, provider_id)
